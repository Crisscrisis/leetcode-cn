## leetcode56-1 验证回问串数组中数字出现的次数

### 思路
```
1.对所有数组内所有元素做一次异或运算，则剩下的是那两个不相同的数做异或运算的结果，因为相同的两个数，做异或运算之后，结果为0，被抵消掉。
2.第一次做完异或运算的结果，是两个不同的数异或后产生的，其中包含了两个不同的数的信息，比如结果为0x00110,则说明右边往左数，这两个数的第二位和第三位是不同的。
3.随便选择一个不同的位，比如说从右边数的第一个不同的位，以此作为一个mask，分别跟数组中的所有数再做一次按位与的比较，如果按位与结果是0，则归入一组，结果非0，
则归入另一组，则这样会把所有元素分为两组，并且两个不同的数分别在这两组，这两组的其余元素，都是一对一对相同的数，对这两组再做一次异或操作，会把那些相同的数过滤掉，
则这两组剩下来的，就是那两个不同的数。

类似题：leetcode136,leetcode137,leetcode645
```

### code
```c++
class Solution {
public:
    vector<int> singleNumbers(vector<int>& nums) {
        int a = 0, b = 0, res = 0;
        for(int i = 0; i < nums.size(); i++) {
            res = res ^ nums.at(i);
        }
        int div = 1;
        while ((div & res) == 0) {
            div <<= 1;
        }
        for(int i = 0; i < nums.size(); i++) {
            if (nums[i] & div) {
                a = a ^ nums[i];
            } else {
                b = b ^ nums[i];
            }
        }
        return {a, b};
    }
};
```
